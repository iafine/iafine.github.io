BOM对象总结

BOM全称叫浏览器对象模型，用于JavaScript访问Web浏览器的功能。本文的目的是总结下基本的BOM使用。

window对象

window是BOM的核心，是一个Global对象，在全局作用域里声明的变量和函数，都可以通过window来访问。比如：

var name = 'iafine'
function sayHello() {
    console.log(this.name)
}

console.log(window.name)    // iafine
sayHello()                    // iafine
window.sayHello()            // iafine

window.open可以打开一个特定的URL，类似于<a>标签，如下：

window.open('https://www.baidu.com')

location对象

顾名思义，提供了当前窗口加载的页面信息。我们以Google页面作为参考。

https://www.google.com/webhp?hl=zh-CN&ictx=2&sa=X&ved=0ahUKEwjmteDX7sbgAhVTObwKHeTNBZQQPQgH

具体用法如下：

location.host    
// "www.google.com"

location.hostname    
// "www.google.com"

location.href
// "https://www.google.com/webhp?hl=zh-CN&ictx=2&sa=X&ved=0ahUKEwjmteDX7sbgAhVTObwKHeTNBZQQPQgH"

location.pathname
// "/webhp"

location.port
// ""

location.protocol
// "https:"

location.search
// "?hl=zh-CN&ictx=2&sa=X&ved=0ahUKEwjmteDX7sbgAhVTObwKHeTNBZQQPQgH"

location.reload可以重新加载当前显示的页面，如果不传递参数，那么页面可能是从缓存中加载的，如果传递参数，并且是true，如location.reload(true)，那么页面会强制从服务器上重新加载。

navigator对象

用于识别客户端浏览器的事实标准。我们以Chrome为例，具体的属性和方法如下：

navigator.appCodeName
//"Mozilla"

navigator.appName
// "Netscape"

navigator.appVersion
//"5.0 (Macintosh; Intel Mac OS X 10_14_2) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/71.0.3578.98 Safari/537.36"

navigator.cookieEnabled
// true

navigator.javaEnabled
// ƒ javaEnabled() { [native code] }

navigator.language
// "zh-CN"

navigator.mimeTypes
// MimeTypeArray {0: MimeType, 1: MimeType, 2: MimeType, 3: MimeType, application/pdf: MimeType, application/x-google-chrome-pdf: MimeType, application/x-nacl: MimeType, application/x-pnacl: MimeType, length: 4}

navigator.onLine
// true

navigator.platform
// "MacIntel"

navigator.plugins
// PluginArray {0: Plugin, 1: Plugin, 2: Plugin, Chrome PDF Plugin: Plugin, Chrome PDF Viewer: Plugin, Native Client: Plugin, length: 3}

navigator.product
// "Gecko"

navigator.userAgent
// "Mozilla/5.0 (Macintosh; Intel Mac OS X 10_14_2) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/71.0.3578.98 Safari/537.36"

navigator.vendor
// "Google Inc."

history对象

histtory对象保存了用户的上网记录。

history.go(-1)    // 后退一页，类似于history.back()
history.go(1)    // 前进一页，类似于history.forward()
history.go(2)    // 前进两页
