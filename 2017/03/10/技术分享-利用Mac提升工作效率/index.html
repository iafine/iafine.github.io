技术分享-利用Mac提升工作效率

这是今年给组内做的分享资料，主题是提升Mac工作效率。

对于工作来讲，Mac应当是效率最高的系统了，不管是写代码还是编写文档，利用好Mac，工作效率会大大提升。这篇文章从快捷键开始，来整理一下那些可以提升Mac工作效率的方式吧。

快捷键

对于多年不用鼠标的人，使用快捷键可以节省好多时间，从最简单的复制粘贴到光标位置移动，只要用的熟练，完全可以抛弃鼠标了，下面是经常用到的一些快捷键。

  • 复制粘贴全选（需要说明的是，Mac和iPhone可以连接起来使用此功能，可以在任意一台设备上进行复制，然后在另外一台上进行粘贴）

    cmd+c    // 复制
cmd+v    // 粘贴
cmd+a    // 全选

  • 切换应用及窗口

    cmd+tab        // 切换应用
cmd+`        // 切换统一应用的窗口

  • 关闭应用及窗口

    cmd+q        // 关闭当前应用
cmd+w        // 关闭当前窗口

  • 新建窗口及标签页

    cmd+n        // 新建窗口
cmd+t        // 新建标签页

  • 移动光标及处理

    ctrl+a        // 光标移动到行首
ctrl+e        // 光标移动到行尾
ctrl+k        // 删除光标右边内容

  • 空格预览

    选中某个文件，直接按下空格键，可直接预览内容。

  • 显示简介

    cmd+i        // 显示简介，查看图片尺寸非常方便。

软件篇

Snip - 定制快捷键工具

Snip是一款可以定制App启动快捷键的工具，默认是根据Dock排列，利用cmd+{Dock位置次序}来启动，当然，你也可以自己定义某个App启动的快捷键，设置也很简单，偏好设置那里直接修改就行。

这款软件比较适合新手使用，如果使用它的话，默认的cmd+1/2/3/4/…这些快捷键就会变成启动某个App，如果你对这些键位还有其他需求，就只能自定义快捷键了。

Snipaste - 截屏软件

对于好多人来讲，一开始接触电脑都是通过QQ来的，QQ的截屏功能在Windows上也是深入人心，换到Mac之后，由于Mac版的QQ体验真心差评，并且工作上一般不会使用QQ，所以截屏这一普通需求一直没找到好的解决方案。

虽然Mac上的截屏软件也很多，但是没有一款体验能和Snipaste一样，给人一种的舒服感。并且这款软件目前还是Beta版，作者是国人，极力推荐大家使用，相信Snipaste一定会在Mac上站稳脚跟。

Usage - 习惯记录

Usage的特点是可以自动追踪你的电脑使用习惯，统计出软件使用时长，并且UI也很直观，如果长期使用的话，获取到的数据会很有价值。

Typora - Markdown写作

Mac上的Markdown写作工具是层出不穷，除了Typora，没有一款给人眼前一亮的感觉，Typora的使用者评价都很高，可以去Google一下，同时最重要的是目前还是免费，还在beta期。

Typora给人的感觉就是使用起来很简洁，写的时候给人一种很享受的感觉，是一款专注于写作，干净纯粹的工具。

提醒事项 - 系统级TODO工具

如果单纯是使用TODO功能，我极力推荐这款系统工具。使用简单，并且免费跨平台，还有什么理由不用它呢？

Alfred

Alfred之所以要单独分开来说，是因为它对Mac使用的影响，实在深远。如果说Mac上有一款软件能称得上是神器的话，那么非Alfred莫属。

Alfred是少数可以称得上全能的App，它的使用范围很广，某些功能可以直接替代一些小型工具类App。

启动APP

输入App名称可以快速启动相关App，这个功能十分常见，比如快速启动VSCode。

  • find命令可以快速找到该文件；
  • in命令可以匹配文本内容，功能强大。

剪切板

这是一个十分实用的模块，可以记录下你之前的历史复制记录，在需要的时候快速地找到它。

Alfred里你可以自定义一个快捷键专门打开该功能，一般我是使用clip命令来开启剪切板。用多了就会觉得十分高效。

代码片段

这也是经常使用的一个功能，可以快速地获取一段文本，然后粘贴到你需要的地方。其实在有些IDE里也是有这个功能，比如Xcode。

同时也是可以自定义一个快捷键打开该功能，一般我是使用snip命令来打开代码片段。

系统功能

这个不是经常能用到，但是Alfred支持此模块。对于一些常用到的系统功能，可以使用命令方式完成。比如：

  • sleep命令可以使电脑睡眠；
  • hide命令可以最小化当前应用；
  • quit命令可以关闭当前应用。

同时还有很多，可以在偏好设置那里去查。
