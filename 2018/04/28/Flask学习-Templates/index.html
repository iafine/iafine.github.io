Flask学习-Templates

Flask使用Jinja2作为默认的模版引擎，Jinja在日本文化里是神社的意思，神社的英文单词temple，和template有点像，所以由此产生。

Jinja2的语法在这里就不过多介绍了，网上有的是，我们主要学习下Flask中Templates的用法。我们以一个blog为例，一共有两个页面，首页和关于页面。

项目搭建

项目创建用到了pipenv，可以分离开发环境。

- templates
--- about.html
--- home.html
--- layout.html
- app.py
- Pipfile
- Pipfile.lock

首先定义好路由：

from flask import Flask

app = Flask(__name__)

@app.route('/')
@app.route('/home')
def home():
    return 'home page'

@app.route('/about')
def about():
    return 'about page'

if __name__ == "__main__":
    app.run(
        debug=True
    )

然后，运行测试一下，看是否可以跑通。

python app.py

模版定义

我们先从home页面出发，调用home路由，返回的是一个简单的html代码，首先在templates里新建一个home.html。用于展示home页面。

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Home Page</title>
</head>
<body>
    <p>Home Page</p>
</body>
</html>

然后修改app.py代码。

from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
@app.route('/home')
def home():
    return render_template('home.html')

@app.route('/about')
def about():
    return 'about page'

if __name__ == "__main__":
    app.run(
        debug=True
    )

运行发现，打开home页面已经变成了一个大字的home page。

当然，about同理。

模版传参

我们在home页面显示两篇文章，目的是将文章集合传递给html显示。首先修改app.py：

from flask import Flask, render_template

app = Flask(__name__)

posts = [
    {
        'author': 'iafine',
        'title': 'Falsk',
        'content': '哈啊哈，哈哈',
        'date_posted': '2018-12-18'
    },
    {
        'author': 'iafine',
        'title': 'Sunny',
        'content': '今天的阳光很好',
        'date_posted': '2018-12-19'
    }
]

@app.route('/')
@app.route('/home')
def home():
    return render_template('home.html', posts=posts)

@app.route('/about')
def about():
    return render_template('about.html')

if __name__ == "__main__":
    app.run(
        debug=True
    )

然后修改home.html，如下：

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Home Page</title>
</head>
<body>
    {% for post in posts %}
        
{{ post.title }}

        
By {{ post.author }} on {{ post.date_posted }}

        
{{ post.content }}

    {% endfor %}
</body>
</html>

运行之后，可以发现参数正常的显示出来了。

接下来把title的判断也加上，如果是首页，显示Flask Blog，其他页面显示Flask Blog-xxx这样。首先修改app.py。

from flask import Flask, render_template

app = Flask(__name__)

posts = [
    {
        'author': 'iafine',
        'title': 'Falsk',
        'content': '哈啊哈，哈哈',
        'date_posted': '2018-12-18'
    },
    {
        'author': 'iafine',
        'title': 'Sunny',
        'content': '今天的阳光很好',
        'date_posted': '2018-12-19'
    }
]

@app.route('/')
@app.route('/home')
def home():
    return render_template('home.html', posts=posts)

@app.route('/about')
def about():
    return render_template('about.html', title='about')

if __name__ == "__main__":
    app.run(
        debug=True
    )

然后修改home.html和about.html的head信息。

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    {% if title %}
        Flask Blog - {{ title }}
    {% else %}
        Flask Blog
    {% endif %}
</head>

判断如果有title，就拼接在后面，否则不拼接。

运行之后，访问about页面应该就可以看到变化了。

提取模版重复代码

至此，基本的逻辑代码已经完成，但是还有点需要优化的是，home.html和about.html的代码会有一些重复的在里面。我们要将重复的代码头部和尾部提取出来。

首先新建个layout.html：

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    {% if title %}
        Flask Blog - {{ title }}
    {% else %}
        Flask Blog
    {% endif %}
</head>
<body>
    {% block content %}{% endblock content %}


```

我们定义了一个block content代码块，用于填充逻辑代码，然后简化`home.html`和`about.html`。

```html
{% extends "layout.html" %}
{% block content %}
    {% for post in posts %}
        
{{ post.title }}

        
By {{ post.author }} on {{ post.date_posted }}

        
{{ post.content }}

    {% endfor %}
{% endblock content %}

修改后的home页面，可以发现，主要剩下核心的逻辑代码。

{% extends "layout.html" %}
{% block content %}
    
About Page

{% endblock content %}

about页面也简单很多。

至此，基本的模版操作就结束了。
